The main differences among var,let,const are 1.var is functional scope.On the other hand, let and const are block scoped. 2.var can be updated and can be declared again.let can be updated but cannot be re-declared.const cannot be updated nor can be declared again. 3.var and let can be declared without initialization. const cannot be declared without initialization.

The differences between regular function and arrow functions are1. 'this' value inside a regular function is dynamic.But 'this' inside the arrow function is bound lexically and equals to 'this' of the outer function. 2.Inside the regular functions it contains the list of arguments.On the other hand, The arrow function, doesn't define arguments.If the arrow function has one expression, then it returns the expression, even without using the keyword.

differences between map,forEach & find,filter are as follows1.forEach() method does not create a new array.map() method creates a new array. 2.forEach() method does not return anything.map() method returns the new array according to the conditions of the provided callback function. 3.find() method is used to find all the descendant elements of the selected element.filter() method is used to filter all the elements. 4.find() method only gives the first element that fulfills the condition.filter() method gives all the element that fulfills the conditions.

Template literals are literals with backtick (`) characters,it allows us to do multi-line code. It is usefull for string interpolation with embedded expressions.That's why it is informally called ase template strings as it is used to create strings by doing substitution of placeholders. We can handle the data dynamically with template strings.
